
# La Ria Portraits (Next.js)

## Lokální spuštění

```bash
npm install
npm run dev
```

Otevřete http://localhost:3000

## Nasazení na Vercel

1. Nahrajte tento projekt na GitHub nebo jako ZIP.
2. Ve Vercel zvolte **Add New → Project** a importujte projekt.
3. Klikněte **Deploy**.
